# rule-based-ai-doctor
AI System Doctor for Linux (C++ Based) - Rule-Based AI

A local AI-powered diagnostic and recommendation tool that:
Analyzes Linux system health,
Detects performance, hardware, and security issues,
Suggests actionable fixes in plain English,
Runs fully offline (privacy-friendly)

Use native Linux sources:
/proc/cpuinfo, /proc/meminfo,
lsblk, df, smartctl,
dmesg, journalctl,
Temperature via /sys/class/thermal,
Motherboard + serial

AI Reasoning Engine: Rule-Based AI (Fast & Deterministic):
Easy, No cloud, Limited learning
